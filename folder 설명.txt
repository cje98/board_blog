[board-front src 하위에 생성한 폴더 구조 설명]
- apis : back과 연결할 api 폴더
- assets : 이미지나 비디오 정적인 요소들을 넣을 폴더
- components : 공통 컴포넌트 작성
- constants : 상수
- hooks : hook 함수
- layouts : header, footer
- mocks : 더미 데이터 관리(mock : 실제 데이터를 모방해서 동작 확인하는 방법)
- stores : 상태 관리
- types : 각종 타입 저장
- utils : 각종 유틸 저장
- views : view, 페이지 관련
